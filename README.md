# AI Meal Planner

## Project Overview
This app generates a personalized meal plan based on a user’s fitness goal, calorie target, dietary preference, and number of meals per day. It uses processed food data to build a meal plan and an AI layer to explain why the selected meals match the user’s needs.

## Supported User Tasks
- Select a goal: weight loss, maintenance, or muscle gain
- Enter calorie target
- Select diet type: vegetarian, high-protein, or balanced
- Select number of meals per day
- Generate a 1-day meal plan
- View calories per meal and total calories
- Get a short AI explanation of the plan

## Out of Scope
- No login/signup
- No live nutrition API
- No medical advice or diagnosis
- No grocery delivery
- No advanced dashboard
- No image generation
- No mobile app
